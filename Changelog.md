# Changelog

All notable changes to this project are documented in this file.

The format is based on https://keepachangelog.com/
and the project follows semantic versioning.

---

## [1.0.0] – Initial Release

### Added
- Arduino Micro based alarm system
- Code-protected alarm arming and disarming
- Alarm latch functionality
- Red blinking TFT display on alarm
- Siren and alarm LED output
- EEPROM persistent storage
- Keypad (TP229) support
- Factory reset and code change
- Complete documentation (wiring, service, operation)

### Tested
- All light barrier inputs
- Alarm latch and acknowledgment
- Power cycle persistence
- Code validation
